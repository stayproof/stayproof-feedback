# StayProof Feedback

Community issue tracker for [StayProof](https://stayproof.co) — the Chrome extension that compares accommodation across Booking.com, Airbnb, and Agoda to help you find genuinely good places to stay.

## How to Report

- **Bug?** → [File a bug report](../../issues/new?template=bug-report.yml)
- **Platform results missing or wrong?** → [Report a platform issue](../../issues/new?template=platform-issue.yml)
- **Feature idea?** → [Request a feature](../../issues/new?template=feature-request.yml)
- **Question?** → [Start a discussion](../../discussions)

## What StayProof Does

StayProof searches Booking.com, Airbnb, and Agoda simultaneously and cross-references results with Google Maps reviews. It detects rating anomalies, deduplicates listings across platforms, and ranks properties by genuine review trustworthiness — not paid placement.

The extension runs entirely in your browser. It does not send your data to any server.

## Scope

This repo is for **community feedback only** — bug reports, feature requests, and discussion. The extension source code is maintained separately.
